プライバシーポリシー

最終更新日: 2025年4月30日

1. 収集する情報

  • 音声データ（マイク入力）
  • テキスト会話内容
  • 検索クエリ
  • 利用状況データ（クラッシュログ、Analyticsなど）

2. 収集方法

ユーザー入力時に当社サーバーへ送信／自動取得。TLSで暗号化。

3. 利用目的

  1. GPT応答生成
  2. 検索結果の要約
  3. サービス改善・障害解析
  4. 法令遵守

4. 第三者提供

以下のサービスに必要最小限のデータを送信します。

  • OpenAI API
  • SerpAPI
  • Apple Speech Framework
  • Firebase Analytics

5. データ保持期間

サーバーログは30日以内に匿名化または削除。

6. ユーザーの権利

開示・訂正・削除は privacy@haruemon.app までご連絡ください。

7. セキュリティ

TLSで通信を保護し、APIキーはサーバー内で安全に管理します。

8. 13歳未満のお子様

13歳未満の子どもを対象としていません。該当する情報は収集しません。
